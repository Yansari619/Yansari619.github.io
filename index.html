Recent Projects

Cede.

Co-Founder & Full-Stack Developer

A campus based startup app designed as a service for students to have their parcels collected from the local post office. Users can request for their parcels to be collected and delivered at their accommodation whilst tracking the deliverer's location. Users can also register to become deliverers and gain an income. Status: Deployment stage

Front-End Development

Designed an appealing user interface with HTML/CSS, implemented the logic and functionality with Typescript and utilised libraries such as react native elements. Used Google Map API's for the location features of the app.

Back-End Development

Used RESTful API's to communicate between front-end and back-end to GET and POST user data when neccessary. Assisted in implementing the Mongoose Schema model and endpoints to create the core functionality of the app. Overall use of a MERN Stack: MongoDB, Express.js, Node.js, React Native.

AutoExport

Front-End App Development

Designed and created a mobile app to optimise car exporting businesses. Admins assign jobs to the car transporters, track their location on a map and view completed jobs through search queries. Drivers accept jobs, categorically take pictures of cars and view their completed jobs. Status: Final stages of development, changing current design, and implementing design of improved features such as search queries and route tracking.

Implementation & Technologies

Responsible for the client-side of the app, using HTML/CSS to allow for an appealing user interface, Typescript to implement the logic for the components and functionality. Pictures of the cars were stored in the S3 (AWS), and the information of each car, location and job was communicated with the server through Google Map API's and RESTful API's. Overall use of the MERN Stack.

Impossible

IOS App Development

Created a mobile app game called 'Impossible 60 seconds' and deployed on the IOS app store. Objective is to avoid all obstacles for the longest time possible. Gained over 600 downloads. Status: Compatible only on old IOS version updates.

Implementation & Technologies

Created the design through XCode's Storyboard UI to ensure compatibility of different phone sizes and used Swift along with the Sprite Kit framework to implement the physics neccessary for the objects of the game to change momentum, direction and speed according to collisions.
